## Heap
### 簡介
Heap，堆積，為一種**樹狀結構**。每個頂點稱為節點(Node)，值儲存在節點中。
* Min heap
<br>所有父節點 **<** 子節點。

![image](https://github.com/HTY62006/MyLearningNote/blob/master/large_image/HS01.png)
* Max heap
<br>所有父節點 **>** 子節點。

![image](https://github.com/HTY62006/MyLearningNote/blob/master/large_image/HS02.png)
